# AI Exam Answer Explainer

An AI-powered system that converts evaluated exam answers into clear, explainable feedback.

## Problem
Students receive marks without knowing why marks were lost. Teachers lack time to provide detailed feedback.

## Solution
The system analyzes a student's answer against a marking rubric and explains mistakes, lost marks, and how to improve.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (FastAPI)
- AI: Large Language Models
- Hosting: Vercel (Frontend), Render (Backend)
